# Advanced CSS3 Responsive Portfolio

## 📌 Project Overview

This project is a modern personal portfolio website developed using HTML5, CSS3, and JavaScript. It demonstrates advanced CSS concepts such as CSS Grid, Flexbox, CSS Variables, Responsive Design, and Dark Mode.

## 🚀 Features

- Semantic HTML5 Structure
- CSS Grid Layout
- Flexbox Layout
- Responsive Design
- Mobile-First Approach
- CSS Variables
- Dark Mode Toggle
- Smooth Scrolling
- Hover Effects
- Responsive Navigation Bar
- Contact Form
- Animations
- Fully Responsive Portfolio

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
